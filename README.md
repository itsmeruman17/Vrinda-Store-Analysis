# 📊 Excel Data Analytics Project – Vrinda Store Sales Performance


## 📌 Project Overview

This project focuses on analyzing sales data of Vrinda Store using Microsoft Excel.  
The goal is to transform raw sales data into meaningful business insights using Excel tools such as:

- Data Cleaning
- Pivot Tables
- Charts
- Interactive Dashboard
- VBA Automation

The final outcome is a dynamic and user-friendly dashboard that supports data-driven decision-making.

---

## 🎯 Objective

- Clean and prepare real-world sales data
- Perform Exploratory Data Analysis (EDA)
- Identify sales trends and customer behavior
- Build an interactive dashboard
- Generate actionable business insights
- Support strategic decision-making

---

## ❗ Problem Statement

Vrinda Store operates across multiple sales channels and states. However, the management faces challenges in:

- Identifying top-performing sales channels
- Understanding customer segments
- Managing high return and cancellation rates
- Detecting state-wise performance variations
- Making quick business decisions using raw data

The project aims to solve these problems by analyzing historical sales data and presenting insights in a structured dashboard.

---

## 📂 Dataset Description

**Dataset:** Raw Sales Data  

### Fields Included:
- Order Date  
- Gender  
- Age  
- Category  
- City  
- State  
- Channel  
- Sales Amount  
- Quantity  
- Order Status  

---

## 🧹 Data Cleaning Process

- Removed blank rows and columns
- Standardized column names
- Corrected data formats (dates, numbers, text)
- Created a separate cleaned dataset sheet
- Validated missing and inconsistent values

---

## 📊 Data Analysis

### 1️⃣ Pivot Table Analysis
Multiple pivot tables were created to answer specific business questions:

- Sales by Channel
- Sales by State
- Sales by Gender & Age
- Order Status Analysis
- Monthly Sales Trends

Each pivot table focuses on structured and clear business insights.

---

### 2️⃣ Dashboard Creation

The interactive dashboard includes:

- Visual charts for sales insights
- Slicers for filtering (Channel, Category, Month, etc.)
- KPI indicators
- Comparative analysis across states and channels

This enables quick and effective decision-making.

---

### 3️⃣ VBA Automation

To improve efficiency:

- Record Macro used
- Refresh Button added
- Reset Slicer Button created

This reduces manual effort and improves usability.

---

## 🔎 Key Business Insights

### 1️⃣ Top Performing Sales Channels

Vrinda Store should focus on:

- Amazon
- Flipkart
- Myntra

These platforms generate the highest revenue and order volume.

---

### 2️⃣ High Demand but High Return States

- Karnataka
- Maharashtra

These states show strong demand but also high returns.

---

### 3️⃣ Most Valuable Customer Segment

**Women (Age 46+)**

- Highest number of orders
- Maximum revenue contribution

---

### 4️⃣ Major Operational Issue

High order returns and cancellations are negatively impacting revenue.

---

## 💡 Recommendations

1. Improve product quality and size accuracy  
2. Focus marketing investment on top-performing channels  
3. Use better courier services in high-return states  
4. Optimize return management strategy  
5. Personalize marketing for high-value customer segment (Women 46+)

---

## 📈 Business Impact

- Higher revenue by focusing on top channels
- Reduced returns through quality improvement
- Improved customer satisfaction
- Faster and smarter decision-making
- Time savings through VBA automation
